Twitter Intents allows you to add Twitter intents functionality to your blog. Easily let your readers reply, retweet, or favorite your posts!

== Description ==

Twitter Intents

== Installation ==

Upload to your plugins/ folder and activate.

After that, you should see a 'Twitter Intents' widget when adding a new Post or Page.

To Use:

1. Publish Post
2. Tweet it!
3. Grab the Tweet ID ( clicking on the tweet or the details link on Twitter )
4. Edit the post and add the Tweet ID under the 'Twitter Intents' field
5. You should see the Twitter icons appear at the bottom of your post
